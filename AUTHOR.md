## Development Notes

This portfolio website represents a comprehensive, custom-built application developed entirely by Samarth Shukla. Every component, design decision, and implementation has been carefully crafted to create a unique and professional showcase.

### Custom Features Implemented

#### Visual & UI
- **Custom Particle Background System** - Advanced constellation system with twinkling stars
- **Hero Asteroid Animation** - Dynamic asteroid particles with dust trails
- **Typewriter Effect** - Custom typing animation for hero section
- **Oneko Cat Integration** - Interactive cursor-following cat
- **Theme System** - Custom dark/light mode implementation
- **Smooth Scrolling** - Lenis integration for premium scrolling experience

#### Backend & APIs
- **Spotify Integration** - OAuth flow with refresh token management
- **Telegram Bot** - Custom bot for contact form notifications
- **Gemini AI Integration** - AI-powered chat interface
- **MDX Blog System** - Custom blog implementation with syntax highlighting
- **API Routes** - Custom Next.js API endpoints for all integrations

#### Technical Implementation
- Next.js 15 with App Router and Turbopack
- TypeScript for type safety
- Tailwind CSS with custom design system
- Shadcn UI components (customized)
- Framer Motion for animations
- Shiki for code syntax highlighting
- Gray-matter for frontmatter parsing
- React Hook Form for forms
- Zod for schema validation

### Development Philosophy

This portfolio emphasizes:
- **Performance** - Optimized builds, efficient rendering, lazy loading
- **User Experience** - Smooth animations, responsive design, accessibility
- **Clean Code** - TypeScript, organized structure, reusable components
- **Modern Stack** - Latest Next.js, React 19, cutting-edge features

### Design Approach

Every design element has been thoughtfully considered:
- Modern, minimal aesthetic
- Cohesive color palette
- Responsive across all devices
- Smooth transitions and micro-interactions
- Professional typography
- Accessible contrast ratios
